The Mandelbrot set is the set of complex numbers c for which the iteration below remains bounded. Its boundary is a fractal — infinite detail at every scale of magnification.
z₀ = 0
zn+1 = zn² + c
Julia mode: click any point on the canvas to set that value as the constant c and render its corresponding Julia set.
